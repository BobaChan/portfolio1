Reflection - Week 2, Day 2

Python and Scratch have many things in common. Many of the ideas we learned in Scratch are visible in Python. For example, variables, loops, functions, and conditionals are seen in both. The only difference would be that we learned different types of each today with Python and we have a broader range of choices because we can write the code instead of relying solely on the blocks that are given in Scratch. A few differences would be the fact that Python code must be typed out while Scratch has blocks to interact with. Also, Python has many commands that one must know and so it makes it easier to have bugs in the program as we type everything out while in Scratch the actions are already there and the coder just has to be sure not to contradict themself in the code or something. Overall the two are very similar in ideas but the actual coding is simpler in Scratch than in Python. Yet both are very fun ways to creat new programs!
